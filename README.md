# 장원준 | 경영지원 인재 포트폴리오

> **wonjunjang.dreamitbiz.com**

데이터 기반 문제 해결과 협업을 통해 조직의 효율을 높이는 경영지원 인재, 장원준의 개인 홍보 웹사이트입니다.

## 주요 섹션

- **자기소개** – 프로필, 학력, 자격, 병역
- **경력** – 대전행복신용협동조합 (1년 8개월) + 시청자미디어재단 (6개월)
- **핵심 강점** – 리스크관리, 업무효율화, 고객응대, 프로젝트관리, 커뮤니케이션, 성실함
- **스킬·자격** – 업무역량, 기술역량, 자격증·어학
- **활동·수상** – 경찰서 감사장, 육군 용사상, 인턴 4건, 사회활동 4건
- **연락하기** – 이메일, 전화, 위치

## 기술 스택

HTML + CSS + Vanilla JS (정적 사이트) / GitHub Pages 배포

## 기능

- 다크/라이트 모드 전환
- 한국어/영어 전환
- 스크롤 애니메이션 (Intersection Observer)
- 모바일 반응형 디자인
